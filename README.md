# Customer Churn Prediction

## Overview
This project focuses on predicting customer churn using machine learning techniques and customer behavioral data.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Matplotlib
- Seaborn

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- XGBoost & Random Forest Models
- ROC-AUC, Precision, Recall, F1-Score Evaluation
- Feature Importance Analysis
